
# Sistema E-commerce Eiser

Este es un proyecto de plataforma de comercio electrónico desarrollado como parte del laboratorio de ingeniería de software. El sistema permite la gestión de productos, visualización de categorías y un flujo de compra optimizado.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (jQuery, Bootstrap).
* **Backend:** PHP.
* **Control de Versiones:** Git y GitHub.
* **Diseño:** Responsive Design (adaptable a móviles y tablets).

## 🚀 Características Principales

* **Gestión de Carrito:** Selección y acumulación de productos en tiempo real.
* **Filtros Avanzados:** Búsqueda por categorías y marcas.
* **Interfaz Galáctica:** Diseño moderno con sliders dinámicos y menús colapsables.
* **Seguridad:** Integración con protocolos de autenticación de GitHub.
